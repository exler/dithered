## Requirements
* Bun 1.1.x

## Getting Started

Install all dependencies:

```bash
bun install
```

Run the development server:

```bash
bun dev
```

## Available Scripts

```bash
# Start development server
bun dev

# Build for production
bun run build

# Preview production build
bun preview

# Run the Biome linter
bun lint
```
